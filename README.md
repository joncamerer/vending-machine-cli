# vending-machine-cli

A command line app meant to serve as the user interface for a vending machine. Built on top of a provided framework to display a menu and handle user options. When run, app will Load available menu from an inventory file and present a navigation menu to the user. User must enter money in order to be able to purchase any item from the vending machine. Will track and audit any purchases and keep inventory updated. Machine will also provide a dated sales file, displaying total sales since the machine was started. Developed with a partner. 

Developed in Java using principles of OOP, File I/O, and JUnit testing.
